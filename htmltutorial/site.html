Aprendendo a fazer um site html básico pelo notepad

Dicas e um pequeno tutorial para fazer seu primeiro site html


Para fazer seu primeiro site em html, deve-se primeiro abrir algum programa editor de texto, como o notepad ou notepad++

• Após isso, deve abrir o "<⠀html >", elemento básico da estrutura do html. Ela conterá todos os elementos do documento

• Abrir "<⠀head >", tag usada como uma especie de cabeçalho, ajudará o navegador a obter informações importantes

• Em seguida, abrir "<⠀title >", esta tag definirá o titúlo da página, estará visivel na aba

• Depois abrir o "<⠀body >, a tag funcionará como o corpo do documento, todos os elementos visíveis estarão dentro dela

• Posteriormente digitar o que você deseja posicionar no seu documento, com tags específicas para funcionalidades específicas

e não se esqueca de fechar as tags usando <⠀/ >


Os códigos do seu primeiro site html já estão prontos, para visualizar em seu navegador, basta salvar como .html e abrir em seu browser
